Projects

Fund4Future

Fund4Future is a crowdfunding platform built to help students and communities raise financial support for their projects and initiatives. The website includes a landing page for public campaigns and a dashboard for managing fundings. To ensure security and transparency, each funding page features a comment section, while a built-in chat system allows communication between users and staff/admins to verify campaigns and prevent fraud. The platform also integrates with Midtrans to handle balance top-ups and transactions.

This project was developed collaboratively using an Agile approach. I contributed as a Scrum Master, Product Owner, and Fullstack Developer, working alongside my teammate to design, build, and refine the platform.

Tech stack: HTML, CSS, JavaScript, Laravel, Bootstrap, MySql

Biodata Guru

Biodata Guru is a web application built to help schools manage and organize teacher information more efficiently. The platform allows users to input teacher data directly through a form or by uploading an Excel template that has been preset for consistency. To support better data management, managers can export the collected information into PDF format, either as a complete list or based on specific filters such as kindergarten, primary, or high school teachers. Additionally, the system provides access to supporting documents (berkas) by listing uploaded PDFs for easy reference. With these features, Biodata Guru simplifies administrative tasks, reduces manual errors, and ensures that teacher data can be stored, viewed, and exported in a structured and professional way.

Tech stack: HTML, CSS, JavaScript, PHP, MySql
